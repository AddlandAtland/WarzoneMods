This mod gives you absurd perms as Game Master.

1. Set owners of any given territory
2. Add or subtract armies from any given territory

How to use?
To find a user's ID, open up their profile on a browser. There, "p=##(ID)##", user ID can be obtained by subtracting the first and last two digits from the profile ID
Example: with "Profile?p=32116011401", the profile ID is 32116011401 --> ##1160114## --> User ID would be 1160114
